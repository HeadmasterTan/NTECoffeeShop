
# ☕ 異環店長特供自動工具

本工具基於`OpenCV`，所以不能掛後台，望周知。

請注意：目前此工具尚無法做到 100% 完美通關，特別是 3-6、3-9 等後續關卡，完美通關概率可能不到 20%，不過本人測試多次嘗試後還是能夠達成完美通關的。

![安魂曲](./docs/Lacrimosa.png)接下來讓 安魂曲 作為代理店長幫你營業。

[點擊下載](https://github.com/HeadmasterTan/NTECoffeeShop/releases)

## 📘 使用說明

* 將 `zip` 解壓後，`以管理員身份` 運行 `NTEFishingTool.exe` 即可。

* 雇員請不要選用 `安魂曲`，因為番茄醬是給她的工資，所以不賣。

* 僅支持遊戲分辨率 `1920x1080` 和 `1280x720`。

* 僅支持 `Windows 10` 和 `Windows 11`。

* 選中 `重複執行關卡`，再選擇對應的關卡後啟動，就會一直循環選中關卡直到都市活力![都市活力](./docs/Energy.png)耗盡，工具會自動停止。
	- 如果你希望快速清空都市活力![都市活力](./docs/Energy.png)，那麼我推薦關卡 `1-1`。

* 選中 `自動通關全部關卡` 後啟動，會檢索尚未完美通關的關卡，如果有則進入關卡嘗試完美通關，連續 3 次未檢索到關卡後工具自動停止。

* 請確保在如下界面後，再啟用本工具。
![店長特供](./docs/ManagerScene.png)

* `自動營業` 後，除非 `暫停營業`，請勿繼續操作電腦，否則影響工具工作。

## ✨ 涵蓋功能

1. 自動循環指定關卡
2. 自動通關未完美通關的關卡
3. 多語言

## ❔ Q&A

### 問：出餐速度太慢

- 雇員選擇 `翳`，天賦升到 3 級，增加耐性條。
- 可以適當降低遊戲畫質或解析度。
- 可能是設置過色彩增強、濾鏡等功能，建議關閉。
- 如果是AMD顯卡，可以嘗試關閉幀生成。
- 如果上述均無法解決，你可能要遺憾告別 `安魂曲` 代理店長。

### 問：為什麼無法出售番茄醬

- 誠如我開頭所說，番茄醬是 `安魂曲` 的工資，不賣。
