
# ☕ 『異環（NTE）』店長特製自動ツール

本ツールは `OpenCV` をベースにしているため、バックグラウンド（裏画面）での動作には対応していません。 予めご了承ください。

⚠️ ご注意： 現時点では、本ツールによる100%のパーフェクトクリア（星3等）は保証できません。特に `3-6` や `3-9` などの後半ステージでは、パーフェクトクリアの成功率が20%未満になる場合があります。ただし、開発者のテスト環境では、複数回挑戦することで最終的にパーフェクトクリアを達成できることを確認しています。

ここからは、![レクイエム](./docs/Lacrimosa.png)レクイエム が代理店長としてあなたの代わりに店舗を営業します。

[最新版をダウンロード](https://github.com/HeadmasterTan/NTECoffeeShop/releases)

## 📘 使用方法・説明

* `zip` ファイルを解凍後、`NTECoffeeShop.exe` を `「管理者として実行」` してください。

* 従業員（スタッフ）には `レクイエム` を選ばないでください。ケチャップは彼女の給料（報酬）であるため、販売対象外となります。

* 対応しているゲーム解像度は `1920x1080` および `1280x720` のみです。

* 対応 OS は `Windows 10` および `Windows 11` のみです。

* 「ステージ周回」にチェックを入れ、対象のステージを選択して起動すると、都市活力![都市の活力](./docs/Energy.png)が尽きるまで選択したステージをループし続け、ツールは自動的に停止します。
	- ![都市の活力](./docs/Energy.png)都市活力を素早く消費したい場合は、ステージ `1-1` の周回をおすすめします。

* `「全ステージ自動クリア（自動通關全部關卡）」` にチェックを入れて起動すると、まだパーフェクトクリアを達成していないステージを検索し、自動で入場してパーフェクトクリアを試みます。未クリアのステージが3回連続で見つからなかった場合、ツールは自動的に停止します。

* 必ず以下の画面（店舗経営画面）が表示されていることを確認してから、本ツールを起動してください。
![店長特製](./docs/ManagerScene.png)

* 「自動営業」が開始された後は、「営業一時停止」 をクリックしない限り、PCの操作を行わないでください。操作を行うとツールの挙動に影響が出る原因となります。

## ✨ 実装機能

1. 指定ステージの自動周回（ループ）
2. 未パーフェクトクリアステージの自動攻略
3. 多言語対応

## ❔ Q&A

### Q：料理の提供スピード（出餐速度）が遅すぎます。

- 従業員に `翳` を選択し、天賦（タレント）レベルを3まで上げて「忍耐ゲージ」を増加させてください。
- ゲームの画質設定や解像度を少し下げてみてください。
- 彩度強調（カラーエンハンス）や画面フィルターなどの機能が有効になっている場合は、オフにすることをおすすめします。
- AMD製のグラフィックボード（GPU）を使用している場合は、「フレーム生成（Frame Generation）」をオフにしてみてください。
- 上記の方法で解決しない場合は、残念ですが代理店長 `レクイエム` の使用を諦める必要があるかもしれません。

### Q：なぜケチャップが売れないのですか？

- 冒頭でも説明した通り、ケチャップは `レクイエム` の給料ですので、販売することはできません。
