
# ☕ 异环店长特供自动工具

本工具基于`OpenCV`，所以不能挂后台，望周知。

请注意：目前此工具尚无法做到 100% 完美通关，特别是 `3-6`、`3-9` 等后续关卡，完美通关概率可能不到 20%，不过本人测试多次尝试后还是能够达成完美通关的。

## 📘 使用说明

* 将`zip`解压后，`以管理员身份`运行`NTECoffeeShop.exe`即可。

* 雇员请不要选用 `安魂曲`，因为番茄酱是给她的工资，所以不卖。

* 仅支持游戏分辨率 `1920x1080` 和 `1280x720`。

* 仅支持 `Windows 10` 和 `Windows 11`。

* 选中`重复执行关卡`，再选择对应的关卡后启动，就会一直循环选中关卡直到都市活力![都市活力](./docs/Energy.png)耗尽，工具会自动停止。
	- 如果你希望快速清空都市活力![都市活力](./docs/Energy.png)，那么我推荐关卡 `1-1`。

* 选中`自动通关全部关卡`后启动，会检索尚未完美通关的关卡，如果有则进入关卡尝试完美通关，连续3次未检索到关卡后工具自动停止。

* 请确保在如下界面后，再启用本工具。
![店长特供界面](./docs/ManagerScene.png)

* `自动营业` 后，除非 `暂停营业`，请勿继续操作电脑，否则影响工具工作。

## ✨ 涵盖功能

1. 自动循环指定关卡
2. 自动通关未完美通关的关卡
3. 支持港澳台、以及国际服

## ❔ Q&A

### 问：出餐速度太慢

- 雇员选择 `翳`，天赋升到3级，增加耐性条。
- 可以适当降低游戏画质或分辨率。
- 可能是设置过色彩增强、滤镜等功能，建议关闭。
- 如果是AMD显卡，可以尝试关闭帧生成。
- 如果上述均无法解决，你可能要遗憾告别`安魂曲`代理店长。

### 问：为什么无法出售番茄酱

- 诚如我开头所说，番茄酱是`安魂曲`的工资，不卖。
